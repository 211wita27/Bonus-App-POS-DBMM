API access (admin login required)
Log in once and reuse the session cookie for API calls.
curl -c cookies.txt -X POST http://localhost:8080/login -d "username=admin@admin" -d "password=admin"
curl -b cookies.txt http://localhost:8080/api/restaurants
curl -b cookies.txt http://localhost:8080/api/restaurants/1/rewards
Other endpoints: /api/purchases, /api/redemptions, /api/accounts/{id}.
